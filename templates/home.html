Note: Vercel Speed Insights is primarily a client-side monitoring tool designed for frontend applications.

To enable full Speed Insights tracking:

  • For Next.js frontend: Install @vercel/speed-insights and import SpeedInsights in your root layout
  • For other frontend frameworks: Install @vercel/speed-insights and call injectSpeedInsights() in your main entry point
  • For plain HTML: Add the script tag: <script defer src="/_vercel/insights/script.js"></script>

For more details, see the Vercel Speed Insights documentation
